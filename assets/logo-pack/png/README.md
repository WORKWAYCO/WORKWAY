# PNG Exports

Use the PNG generator tool to create raster versions:

```
open ../../../logo-to-png.html
```

Or serve locally for full functionality:
```
cd ../../..
python -m http.server 8000
# Then visit http://localhost:8000/assets/logo-to-png.html
```

## Available Sizes
- 256×256 (small icons)
- 512×512 (standard)
- 1024×1024 (high-res)
